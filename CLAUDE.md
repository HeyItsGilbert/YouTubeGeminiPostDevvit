## Design Context

The only UI surface in this repo is `preview-site/` (React + Vite). Design work
there is governed by two root docs:

- **PRODUCT.md** — strategic: register (`product`), users (subreddit mods),
  purpose, brand personality (quiet/trustworthy/precise), anti-references, and
  the WCAG 2.1 AA bar. Read before any UX/strategy decision.
- **DESIGN.md** — visual system: North Star "The Proof Sheet", paper/ink palette
  with status-only color, single system sans, flat-by-default elevation, and
  component specs. Read before any visual/styling change. The
  `.impeccable/design.json` sidecar carries renderable component snippets and
  tonal ramps.

These were generated by the `impeccable` skill; keep them current if the design
drifts.
